## Key Features

- **Multi-role System**: Admin, Compliance, IFA, and Deal Originator roles
- **Compliance-First**: Built-in approval workflows and audit trails
- **Private Securities Marketplace**: List, browse, and transact private securities
- **Tranche Management**: Support for multiple tranches per security with variable pricing
- **Order & RFQ Workflows**: Complete order management and request-for-quote system
- **AI Agent System**: Automated security collection and compliance review
- **Client Management**: IFA client profiles and broker mappings

## Technology Stack

- **Frontend**: Next.js 14+ (App Router), Tailwind CSS, ShadCN UI
- **Backend**: Node.js with Fastify, PostgreSQL
- **Database**: Supabase Postgres
- **Auth**: Supabase Auth
- **Agent**: Python with OpenAI Agent Kit
